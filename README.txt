Projet Git
Nous allons dans ce projet tester nos connaissances de GIT. 
Pour celà, nous allons créer un fichier README.txt avec le contenu du projet. 
Nous allons ajouter un fichier Text.doc avec un texte. 
Et pour finir nous allons ajouter une fichier script contenan  "Hello World"
Nous aurons des commits à chaque nouvelles modifications: 
   - Création de README.txt
   - Ajout du texte de présentation dans README.txt 
   - Création et moficiation du document text.doc
   - Création d'un fichier script avec "Hello World"
